Protocolos y Consultas

¿Qué incluye el mantenimiento preventivo estándar? expand_more

Incluye revisión sistemática de componentes críticos, lubricación, calibración de sensores y actualización de software operativo. Se emite un certificado de estado tras cada visita.

¿Cómo garantizan el tiempo de respuesta de 12h? expand_more

Mediante un sistema de retén técnico geolocalizado. El plan Pro garantiza un slot de intervención prioritaria que desplaza tareas no críticas automáticamente en nuestro ERP.

¿Es posible integrar sus reportes en nuestro ERP propio? expand_more

Sí, disponemos de una API REST para clientes con planes Custom que permite la sincronización automática de partes de trabajo y consumos de material directamente en su base de datos corporativa.
